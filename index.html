About 项目说明
PinchBench-Upgraded is an independent re-aggregation layer over the public PinchBench dataset. The pipeline runs in five steps: fetch → inventory → scope → task-rankings → category-rankings. See README for methodology and metric definitions. PinchBench-Upgraded 在 PinchBench 公开数据之上做二次聚合，流水线分五步：fetch → inventory → scope → task-rankings → category-rankings。指标口径与方法论详见 README。
