任务排行榜

本榜单数据来源于 PinchBench，面向 OpenClaw 智能体场景衡量主流大模型的整体能力。本榜单在其公开数据基础上做了数据清洗与二次处理：针对原榜单的版本管理混乱、数据缺失等问题，以单次提交（submission）为最小单位，重新做有效性二元判定。此项修正会导致榜单排名产生变化，但绝没有改变任何模型的原始提交数据。将通过校验的多次提交累计摊算到任务级、大类级排行榜，并补充了参数量、时效比等指标。同时，对模型补充国别与开源属性标注，支持按条件筛选模型、按各指标排序。感谢 PinchBench 团队开放数据。
参考链接：PinchBench 任务详情 · PinchBench 数据来源
Data for this leaderboard is sourced from PinchBench, which measures the overall capability of mainstream large models in the OpenClaw agent scenario. We perform data cleaning and secondary processing on its public data: to address issues such as version-management inconsistency and missing fields in the upstream leaderboard, each submission is re-validated as the atomic unit with a binary check. This validation step changes the resulting ranking, but never modifies any model's original submission data. The valid submissions are then amortized into task-level and category-level rankings, with additional metrics such as parameter size and a value (cost-efficiency) score. Country and open-weight attributes are also annotated, with support for filtering models and sorting by any metric. Thanks to the PinchBench team for releasing the data.
